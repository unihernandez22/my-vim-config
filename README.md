# My Vim Config

## Usage
Backup you current config:  
`mv ~/.vim ~/.vim_back`

<br>

Clone the GitHub repository:  
`git clone https://github.com/unihernandez22/my-vim-config.git ~/.vim`

<br>

Open Vim (it will throw an error, press return)

<br>

Install plugins:  
`:PlugInstall`

<br>
